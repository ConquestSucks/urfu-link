# Architecture Documentation

This directory contains architecture artifacts for URFU Link bootstrap.

- `c4-context.md`: Context-level interactions (actors and external systems)
- `c4-container.md`: Container-level view (gateway, services, data, integration)
- `current-state-gap-analysis.md`: Current implementation gaps relative to the target platform architecture
- `domain-boundaries.md`: Bounded contexts and ownership
- `data-conventions.md`: Datastore ownership and naming rules
- `eventing-conventions.md`: Kafka/gRPC integration conventions
- `sequences/call-session-bootstrap.md`: Sequence for call bootstrap flow
- `adr/*`: Architecture Decision Records
- `runbooks/*`: Operational runbook skeletons
